Descripción: Este proyecto automatiza el proceso completo de compra en la tienda de demostración saucedemo.com 
utilizando Python y Selenium. El bot simula el comportamiento de un usuario real: inicia sesión, agrega productos al carrito, 
realiza el checkout y finaliza la compra. Se implementan opciones avanzadas para evitar interferencias del navegador 
(pop-ups, sincronización de cuenta, etc.) y asegurar la ejecución fluida del proceso.

Características:

Automatización del navegador Google Chrome.

Manejo de pop-ups y ventanas emergentes con opciones personalizadas.

Simulación completa de compra con ingreso de datos personales.

Uso de webdriver-manager para evitar instalación manual del driver.
